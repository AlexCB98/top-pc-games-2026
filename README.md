# Top PC Games 2026

A simple HTML project about some of the most anticipated PC games from the second half of 2026.

The page includes monthly game sections, short descriptions, notable releases and a personal ranking.

---

## Features

- Games organized by month
- Short descriptions for each main game
- Lists of other notable releases
- Personal top 5 ranking
